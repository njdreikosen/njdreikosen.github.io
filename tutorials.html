  • Local App Server

    There are two components to my Local Application Server - the 3D printed case and the React app. You can create the entire ensemble, use only one of the case or the app, or use whatever bits and pieces of each you want! Each component can be downloaded from their respective repositories, where the instructions for setting up that component can be found. Links to both repositories can be found below.
